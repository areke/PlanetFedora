About

Fedora Planet is a website that displays blog posts from various Fedora contributors. While these blog posts don't necessarily reflect the opinions of the Fedora Project, they do express the viewpoints of the individuals contributing to Fedora Planet. This application for Android devices was written to make it faster and easier to view the contents of Fedora Planet. In order to do this, it uses the project's RSS feed.

The App

The application combines a beautiful UI with great functionality in order to make the perfect app. It does this all while keeping the interface simple, which allows anyone to use it. And I mean anyone. The application renders perfectly in both portrait and landscape mode to give you the opportunity to choose whichever one you like best.

Cool Features

  • The app includes a feature that makes sharing easy. What is it? A share button. This button allows you to share the article with anyone you want easily.

  • The application also includes a favorites section. This section allows you to view your favorite articles with the touch of a button and to easily add and remove your favorites.
